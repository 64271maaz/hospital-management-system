# 🏥 Hospital Management System

Complete full-stack Hospital Management System built with Node.js, MongoDB and JavaScript.

## 🚀 Live Features
- Patient Management (Add, Edit, Delete, Search)
- Doctor Management (Profiles, Specializations)
- Appointment Booking System
- Billing & Invoice Generation with Print
- Pharmacy & Medicine Inventory
- Laboratory Test Management
- Ward & Bed Management
- Reports & Analytics with Charts
- JWT Authentication (Login/Register)
- Professional Dashboard with Real-time Stats

## 🛠️ Tech Stack
- **Backend:** Node.js + Express.js
- **Database:** MongoDB + Mongoose
- **Frontend:** HTML5, CSS3, JavaScript
- **Auth:** JWT + bcryptjs

## ⚙️ Installation
```bash
cd backend
npm install
npm run dev
```
