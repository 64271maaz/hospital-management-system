# 🏥 Hospital Management System

A complete full-stack Hospital Management System built with 
Node.js, Express, MongoDB and HTML/CSS/JavaScript.

---

## 🚀 Features

- ✅ Patient Management (Add, Edit, Delete, Search)
- ✅ Doctor Management (Profiles, Specialization)
- ✅ Appointment Booking System
- ✅ Billing & Invoice with Print
- ✅ Pharmacy & Medicine Inventory
- ✅ Laboratory Test Management
- ✅ Ward & Bed Management
- ✅ Reports & Analytics with Charts
- ✅ JWT Authentication (Login/Register)
- ✅ Forgot Password with OTP
- ✅ Professional Dashboard with Real-time Stats
- ✅ Responsive Sidebar Navigation

---

## 🛠️ Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- bcryptjs

### Frontend
- HTML5 / CSS3
- Vanilla JavaScript
- Font Awesome Icons
